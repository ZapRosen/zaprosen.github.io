Sphinx mit reStructuredText¶

Erste Schritte¶

Sphinx ist ein Dokumentationsgenerator oder ein Tool, das eine Reihe von Klartext-Quelldateien in verschiedene Ausgabeformate übersetzt und automatisch Querverweise, Indizes und Hyperlinks. erstellt. Das heißt, wenn Sie ein Verzeichnis haben, das eine Reihe von reStructuredText- oder Markdown-Dokumenten enthält, kann Sphinx eine Reihe von HTML-Dateien, eine PDF-Datei (über LaTeX), Manpages und vieles mehr generieren.
